# 📊 CodeAlpha_UnemploymentAnalysis

A data analysis project on unemployment in India from May 2019 to Nov 2020 using Python.

## 📌 Objective
To analyze unemployment rate trends and investigate the impact of COVID-19 on jobs across Indian regions using publicly available data.

## 📁 Dataset
- **Source:** [Kaggle - Unemployment in India](https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india)
- Cleaned file: `data/clean_unemployment.csv`

## 🧰 Tools Used
- Python, Pandas, Seaborn, Matplotlib
- Jupyter Notebook
- Power BI (for dashboard version)

## 📈 Key Insights
- Unemployment spiked in April–May 2020 due to COVID-19 lockdowns.
- Urban areas were more impacted than rural.
- Labour participation dropped drastically during the pandemic.

## 🧠 Policy Recommendations
- Urban job programs and rural employment diversification
- Skilling initiatives for MSMEs and digital roles
